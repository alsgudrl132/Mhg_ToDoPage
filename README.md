# 칸반 스타일 투두 리스트

## 사용된 기술 스택

- Next.js 14.2.24
- Tailwind CSS 3.4.1
- shadcn/ui components
- Zustand 5.0.3
- Lucide React 0.475.0

## 구현된 요구사항

### 보드 기능

- 보드 생성
- 보드 수정
- 보드 삭제
- 보드 순서 변경 (드래그 앤 드롭)

### 할 일 기능

- 텍스트 박스를 가진 할 일 생성
- 할 일 수정
- 할 일 삭제
- 할 일 위치 변경
- 보드 내에서 순서 변경
- 다른 보드로 이동

### 데이터 저장

- LocalStorage를 통한 데이터 유지
